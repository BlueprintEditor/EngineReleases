# Blueprint Engine Releases
## What is Engine?
Engine is the backend plugin for Minecraft 1.21.10 that will parse all modules you create and then allow you to use them inside of your server.

## What is this for?
This is a repository meant to host the downloads for Blueprint software to fetch the latest version of Blueprint Engine.

## What does Blueprint Engine support?
Blueprint Engine supports Paper Minecraft servers running version 1.21.10.
